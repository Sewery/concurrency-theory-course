# Concurrency Theory Course

This repository contains solved tasks from Computer Science laboratories at AGH

## Lab 1

Study of threads and synchronization in Java

## Lab 2

An introduction to the analysis and validation of algorithms describing concurrent processes.

## Lab 3

Synchronization of threads in Java - condition wait and problem of limited buffer

## Lab 3a

Theoretical possibilities of thread interleaving and the race problem

## Lab 4

Comparison of different solutions to the five philosophers' problem in the multithreaded programming paradigm (Java)

## Lab 5

Applying Trace Theory to Thread Scheduling

## Lab 6

Implement and compare a performance of Single Locked List and Fine Grained Locked List

## Lab 7

Implementation of a parallel Gauss-Jordan elimination algorithm for solving systems of linear equations on the GPU platform

## Lab 14, 15

Familiarizing with the CSP concept and solving the limited buffer problem using software in Kotlin and Java
